# KBS Retextures

A DayZ retexture mod featuring custom skins for clothing and shoes.

## Included Retextures

### Base Game
- **Sneakers** - Pink, Black Rose, Cloud, Green, Heart, Jeans, Smile

### Windstride's Clothing Pack (requires [Windstride's Clothing Pack](https://steamcommunity.com/sharedfiles/filedetails/?id=1797720064))
- **Shabby Hoodie** - Thrifted Floral
- **Skinny Jeans** - Floral, Stars

## Installation

1. Subscribe to the mod on the [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=XXXXXXXXX).
2. Enable the mod in the DayZ Launcher.
3. Add the classnames to your server's loot tables or trader files as needed.
